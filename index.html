Projects

My Projects

  • Project 1: USD/ZWG Exchange Rate Monitor

    Developed a real-time exchange rate tracking application using Streamlit, designed to monitor and display the exchange rate between the US Dollar (USD) and the Zimbabwean Dollar (ZWG). The application provides an interactive and user-friendly interface where users can access up-to-date exchange rate data, visualize trends, and explore historical rates.
  • Project 2: AbodeLLM

    Developed an innovative Offline AI Assistant specifically designed for Android devices, utilizing the advanced capabilities of the LLaMA model. This project highlights my enthusiasm in artificial intelligence, mobile app development, and user privacy.
  • Project 3: Forex Prediction using LSTM

    While learning about deep learning and real-time data processing, I developed a system leveraging LSTM networks to predict forex price movements. The model analyzes historical data and technical indicators such as EMA, MACD, and RSI. Operating in real time, it fetches live data, makes predictions, and visualizes actual vs. predicted prices.
  • Project 4: Rchat: A Random Chat App

    Developed a web-based chat application to facilitate anonymous conversations among users. The purpose of the application is to provide an accessible platform for individuals to connect without the barriers of identity, promoting open dialogue and reducing social anxiety. By randomly matching users, the app encourages spontaneity and the exploration of diverse perspectives, fostering genuine connections that may not occur in traditional chat environments. Users can join without revealing personal information, allowing for authentic interactions. This project was developed using Glitch.
  • Project 5: Zim ID Validator

    Developed a JavaScript tool that checks the validity of Zimbabwean national IDs. The tool is designed to be easy to use and integrates smoothly into websites, providing clear messages about whether an ID is valid.
